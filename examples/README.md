# Non-evidentiary examples

Files in this directory demonstrate schema shape and validator behavior. They are synthetic fixtures, not completed experiments, research findings, measured results, registered mechanisms, replications, or evidence for any scientific claim.

An example must use conspicuous placeholder identifiers and must not be copied into a registry without replacing every placeholder, performing the declared work, and recording traceable artifacts. The foundational finding fixture preserves an intentional `NOT_RUN` disposition; it does not imply that an analysis was executed.
