# Research templates

These templates make the Program Charter's reporting obligations explicit. Contributors should retain applicable sections, mark genuinely inapplicable sections with a reason, and avoid deleting an obligation merely because the relevant evidence is unavailable. The **Future CorpusStudio Integration Implications** block is the exception: remove it entirely when the artifact has no CorpusStudio implication.

Completing a template does not advance an evidence state. Claims still require linked, reviewable evidence records for the exact mechanism version and scope.
