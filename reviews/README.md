# Reviews

This directory preserves scoped program-boundary, contract, security, and evidence reviews.

A repository review is not automatically an independent validation. Each review must identify its scope, reviewer relationship, method, findings, unresolved obligations, and disposition. Evidence-state decisions belong in immutable evidence records and remain subject to [LCMRP governance](../GOVERNANCE.md).

- [M0 boundary review — 2026-07-20](M0_BOUNDARY_REVIEW_2026-07-20.md) is an internal adversarial review of the initial governance and reproducibility foundation.
